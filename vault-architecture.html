asktown-pf Vault Architecture

Secure background data gathering with application-level encryption at rest.

Defense in Depth

  • • Application-level encryption using Libsodium
  • • Unique nonces per transaction record
  • • Tokens stored in isolated file paths
  • • Separation of Cron write vs Web read keys

Cron Pipeline

  • • Automated heartbeat via Hermes Cron
  • • Transaction-by-transaction deduplication
  • • Error notification to terminal/logs
  • • Rate-limited TrueLayer polling

Supabase Storage

  • • Encrypted blob storage in Postgres
  • • Metadata indices for date/user (plain)
  • • Row Level Security (RLS) enabled
  • • Managed backups & snapshots
